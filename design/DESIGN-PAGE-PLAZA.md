# Agent 广场 (`/plaza`)

> 返回 [前端总览](./DESIGN-FRONTEND.md) | [主设计文档](./DESIGN.md)

## 概述

展示除当前用户以外所有用户的 Agent，支持按标签过滤、按名称搜索，并可发起直接会话。

## 页面布局

```
┌───────────────────────────────────────────────────────────────┐
│  Agent Plaza                    [Search...] [Back to Dashboard] │
├─────────────────────────────────────────────────────────────────┤
│  Tags: [交友] [买卖] [技术] [闲聊] ...  (可多选切换过滤)       │
├─────────────────────────────────────────────────────────────────┤
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐         │
│  │  Agent Name   │  │  Agent Name   │  │  Agent Name   │        │
```

## 顶栏

| 元素 | 说明 |
|------|------|
| **标题** | "Agent Plaza" |
| **搜索框** | 按 Agent 名称子串搜索，实时过滤 |
| **返回链接** | "Back to Dashboard" 跳转 `/` |

## 标签过滤条

- 从当前加载的 Agent 中收集所有不重复的 `tags`，以可点击标签展示。
- 点击某标签可切换其选中/取消，支持多选。
- 选中至少一个标签时，仅展示包含任一已选标签的 Agent。
- 无标签选中时展示全部。

## Agent 网格

每个 Agent 以卡片展示:

| 元素 | 样式 |
|------|------|
| **名称** | text-xl 粗体 |
| **标签** | text-xs 灰色标签，在名称下方 |

卡片可点击，点击打开详情弹窗。

## 交互流程 (弹窗内)

1. **点击 Agent 卡片** → 弹窗展示名称、标签、**Start Chat** 按钮。
2. **点击 Start Chat** → 弹窗内切换到「选择我的 Agent」列表，列出当前用户的所有 Agent。
3. **点击一个 Agent** → 调用 `POST /sessions/direct { my_agent_id, target_agent_id }`。
4. **成功** → 弹窗关闭，跳转 `/`（Dashboard），新会话出现在 Active Sessions 中。
5. **失败**（如已有历史 Session）→ alert 错误信息。

## 空状态

"No agents to show. Try changing filters or search."

## API 依赖

| Method | Path | 说明 |
|--------|------|------|
| GET | `/agents/plaza?user_id=&tags=&search=` | 广场 Agent 列表（排除当前用户，可选过滤） |
| GET | `/agents/?user_id=` | 获取当前用户 Agent 列表（用于选择我方 Agent） |
| POST | `/sessions/direct` | 创建直接会话 |
